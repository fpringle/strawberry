Detailed Description

The main namespace for the core chess engine.

The namespace chessCore contains all the classes, functions, constants etc used by the core chess engine.

◆ init() [2/2]

Initialise some global constants.

Initialise a the 2-D array of bitboards rays, and the hash keys. The rays are generated by init_rays, and the hash keys by init_keys(). The only difference to init() is that the seed for the hash keys can be specified, for debugging. See move.h and hash.h.
