Introduction

Strawberry is a Chess engine, GUI and interface, all written in C++. It can be played on the command-line, graphically in a GUI, or using the Universal Chess Interface (UCI) protocol.

Strawberry contains 3 modules, each of which is its own repository.

Core engine

The core of Strawberry is the engine, found in src/core.

Graphical user interface (GUI)

Strawberry has a GUI that uses Qt 5 to let the user play on the screen, found in src/gui.

Universal chess interface (UCI)

The Universal Chess Interface (UCI) is a standardised communication protocol that allows chess engines like Strawberry to communicate with user interfaces. The submodule is found in src/uci.

Installation

First you'll need to download the source code: 

 ~$ git clone --recurse-submodules https://github.com/fpringle/strawberry.git
 ~$ cd strawberry/

Building the engine and GUI

Install dependencies

All the submodules rely on Qt 5: 

 ~/strawberry$ sudo apt-get install qt5-default qt5-qmake qtbase5-dev-tools qt5-doc libqt5svg5 libqt5svg5-dev

Build Makefiles using qmake

 ~/strawberry$ qmake

Build Strawberry

 ~/strawberry$ make

Building the documentation

If needed, you can build the documentation locally:

Install doxygen

 ~/strawberry$ sudo apt-get install doxygen

Run doxygen to build documentation

 ~/strawberry$ doxygen
 ~/strawberry$ xdg-open docs/html/index.html

Running Strawberry

To play on the command-line: 

 ~/strawberry$ ./src/core/build/strawberryCLI

To play with the GUI: 

 ~/strawberry$ ./src/gui/build/unix/strawberryGUI
