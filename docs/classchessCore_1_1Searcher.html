Perform a quiescence search on a position.

Quiescence search is a technique to mitigate the horizon effect, where we recursively search through all the upcoming capture tradeoffs to get a better heuristic value for a given position.

Parameters
bThe board to analyse.
alphaThe current value of alpha in negamax search.
betaThe current value of beta in negamax search.
Returns
The heuristic value for the node.
